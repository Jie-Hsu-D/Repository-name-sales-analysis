# Sales Data Analysis with Pandas

## Objective
Analyze sales performance by region and identify top-selling products using Python and Pandas.

## Skills Demonstrated
- Data cleaning and handling missing values
- Aggregation with `groupby`
- Identifying top values with `idxmax`
- Sorting and ranking
- Data visualization with Matplotlib and Seaborn
- Formatted output and reporting

## Key Insights
- West/North region generated highest revenue
- Top-selling product is Laptop, contributing 42.86% of total revenue
- Products ranked by units sold to identify inventory priorities

## Data
- `Sales_data.csv`: Contains product, units sold, unit price, and region information
